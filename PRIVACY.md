# Privacy Policy — GoPro Stitch Uploader

This is a personal-use, single-user command-line tool. It is not distributed
and has no users other than its developer.

This tool does not collect, store, transmit, or share any personal data
belonging to any third party. It reads video files from the developer's own
GoPro SD card and uploads them, via the YouTube Data API, only to the
developer's own YouTube channel, using the developer's own Google OAuth
credentials stored locally on the developer's own machine.

No data is shared with any third party. No analytics, tracking, or
advertising is present in this tool.
